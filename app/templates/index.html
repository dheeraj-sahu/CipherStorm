3-Layer Hybrid Architecture

Robust anomaly detection combining machine learning and rule-based logic for comprehensive protection

Layer 1: Global ML Model

Generalized detection trained on large multi-user datasets

  • Random Forest algorithm
  • Detects common fraud patterns
  • Flags known attack behaviors
  • Learns from historical data

Layer 2: Rule-Based Checks

Real-time heuristics for zero-day fraud detection

  • Transaction limit breach (>95% of user limit)
  • Geographic location mismatch
  • UPI ID validation via Paysprint API
  • Immediate alert generation

Layer 3: User-Specific Analysis

Personalized detection based on individual transaction history

  • Rule-based prediction using user transaction records
  • Adapts to individual user patterns
  • Percentile-based anomaly detection (98th, 90th, 85th, 80th, 70th)
  • Weighted rule system for behavioral outliers
